# AI Agents Control Panel

A small Angular web application built to demonstrate Angular fundamentals.

## Features

- Dashboard page
- Agents list
- Add new agent
- Edit agent
- Delete agent
- Agent details page
- Form validation
- Success messages
- Local storage persistence
- Basic responsive UI

## Angular Concepts Practiced

- Standalone components
- Routing
- Dynamic routes
- Services
- Dependency injection
- Interfaces
- Template-driven forms
- Two-way binding
- Event binding
- Conditional rendering
- Local storage

## Pages

- `/` - Dashboard
- `/agents` - Agents list
- `/agents/new` - Add agent
- `/agents/:id` - Agent details and edit
